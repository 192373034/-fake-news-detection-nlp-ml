# Fake News Detection Using NLP and Supervised Machine Learning Techniques

## 📌 Project Overview

This project focuses on detecting fake news using Natural Language Processing (NLP) and supervised machine learning techniques. The system classifies news articles as real or fake based on textual content.

---

## 🚀 Features

* Text preprocessing using NLP techniques
* Stopword removal and text cleaning
* Feature extraction using TF-IDF
* Supervised machine learning model (Naive Bayes)
* Model evaluation using accuracy, precision, recall, and F1-score

---

## 🛠️ Technologies Used

* Python
* Google Colab / Jupyter Notebook
* Pandas, NumPy
* Scikit-learn
* NLTK

---
